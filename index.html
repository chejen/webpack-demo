Demo 4 - Production Settings

This example uses Entry1.js in "Demo 1" and Entry2.js in "Demo 2" to demonstrate how to optimize our web applications. There are couples of key points:

  • Set process.env.NODE_ENV 'production' and use 'UglifyJsPlugin' to minimize static files.
  • Use 'CommonsChunkPlugin' to extract common modules from multiple entry chunks to commons chunk.
  • If you want to use some libraries from CDN, set 'external' option.
  • Use 'devtool' option to genetate SourceMaps to enhance debugging.

(Run npm run demo4. See index4-entry1.html, index4-entry2.html and their page sources.)
